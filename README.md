# DartMate – iOS Darts Scorekeeper

## Projektbeschreibung
DartMate ist eine iOS-Anwendung zur Dokumentation und Analyse von Darts-Spielen. 
Entwickelt im Rahmen des Moduls DLASMD02 (Projektbericht) an der IU Internationale Hochschule.

## Technische Voraussetzungen
- Xcode 15 oder höher
- iOS 17 SDK
- Swift 5.9

## Build-Anleitung
1. Repository klonen: `git clone https://github.com/KNicoIU/DLASMD02DartMate.git`
2. Projekt in Xcode öffnen: `DartMate.xcodeproj`
3. Zielgerät auswählen (Simulator oder physisches iPhone)
4. Build ausführen: ⌘+R
